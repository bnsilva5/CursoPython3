Ciencia de datos y matemáticas

Numpy: Una biblioteca que agrega soporte para matrices y matrices grandes y multidimensionales, junto con una gran colección de funciones matemáticas de alto nivel para operar en estas matrices.

OpenCV : proporciona una biblioteca, herramientas y hardware optimizados de Computer Vision en tiempo real. También es compatible con la ejecución de modelos para Machine Learning.

Tensorflow : plataforma de código abierto de extremo a extremo para el aprendizaje automático.

Keras : biblioteca de Redes Neuronales de Código Abierto escrita en Python. Es capaz de ejecutarse sobre TensorFlow, Microsoft Cognitive Toolkit o Theano.​ Está especialmente diseñada para posibilitar la experimentación en más o menos poco tiempo con redes de Aprendizaje Profundo.

Sonido

La manipulación de sonido se realiza fácilmente en Python, gracias a algunos módulos muy útiles.

pySonic : una envoltura de Python alrededor de la biblioteca de sonido FMOD multiplataforma de alto rendimiento.
	http://pysonic.sourceforge.net/

PyMedia : un módulo de Python para manipulaciones de archivos WAV, MP3, Ogg, AVI, DivX, DVD, CD-DA, etc. Le permite analizar, demultiplexar, multiplexar, decodificar y codificar todos los formatos compatibles. Se puede compilar para Windows, Linux y Cygwin.
	http://pymedia.org/

PMIDI : la biblioteca PMIDI envuelve la biblioteca de flujos MIDI de Windows para su uso en Python. Con PMIDI, los desarrolladores pueden generar secuencias musicales sintetizadas sobre la marcha en su código para su reproducción para los usuarios.
	http://www.cs.unc.edu/Research/assist/developer.shtml

Mutagen : un módulo para manejar metadatos de audio. Es compatible con archivos de audio FLAC, M4A, Musepack, MP3, Ogg FLAC, Ogg Speex, Ogg Theora, Ogg Vorbis, True Audio y WavPack.
	https://github.com/quodlibet/mutagen

Puertos USB y serie

PyUSB pretende ser un módulo de Python fácil de usar para acceder a dispositivos USB. PyUSB se basa en una biblioteca de sistema nativa para el acceso USB. Actualmente, funciona de inmediato con libusb 0.1, libusb 1.0, libusbx, libusb-win32 y OpenUSB, y funciona con cualquier versión de Python a partir de 2.4, incluidas las versiones de Python 3.
	https://pyusb.github.io/pyusb/

PySerial : biblioteca de acceso al puerto serie de Python.
	https://github.com/pyserial/pyserial

USPP es un módulo Python multiplataforma para acceder a los puertos serie. Por el momento, solo funciona en Windows.
	https://sites.google.com/site/ibarona/uspp
